# Version 8: XStyled and styled-component upgrades

Welcome UI V8 upgrade `@xstyled` to `v4` and `styled-components` to `v6`. This upgrade is a major version because it includes breaking changes.

![Version 8 illustration](todo)

## Upgrade steps

### 1. Upgrade your dependencies

Upgrade `@xstyled/styled-components` and `styled-component` packages:

```bash
yarn upgrade @xstyled/styled-components@^4.0.0" styled-components@^6.1.5
```

### 2. Remove useless dependency

Remove `@types/styled-components` package:

```bash
yarn remove @types/styled-components
```

### 3. Script to migrate

In your project, run this script to

- move typescript on styled component

```diff
+ export const Component = styled.divBox<{ property: string }>(
+ ({ spropertyize }) => css`
- export const Component = styled.divBox(
-   ({ property }: {property: string }) => css`
```

- transform system with xBox component

```diff
+ export const Component = styled.spanBox`
- export const Component = styled.span`
-   ${system}
```
