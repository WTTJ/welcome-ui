# Usage with form libraries

You'll probably use [Final Form](https://github.com/final-form/react-final-form), [Redux Form](https://github.com/erikras/redux-form) or [Formik](https://github.com/jaredpalmer/formik) to manage your form state. If so, make sure you import `Field` from `final/Field`.
