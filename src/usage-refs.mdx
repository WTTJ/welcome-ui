# Refs

You can create a `ref`\* and pass it to a component to access the underlying DOM element.

Note: `ref`s can be created with either `React.createRef()` or `React.useRef(null)` for functional components or a callback ref if using a React `class`.

<Playground>
  {() => {
    const connectedInputRef = createRef()
    const normalInputRef = useRef(null)

    handleConnectedClick = e => {
      connectedInputRef.current.focus()
    }

    handleNormalClick = e => {
      normalInputRef.current.focus()
    }

    return (
      <Form
        onSubmit={() => {}}
        render={({ handleSubmit }) => (
        <form onSubmit={handleSubmit}>
          <ConnectedField name="firstName" component={InputText} label="Connected" ref={connectedInputRef} mb={20} />
          <Field name="lastName" component={InputText} label="Normal" ref={normalInputRef} mb={20} inputRef={normalInputRef} />
          <Button onClick={handleConnectedClick}>Click to focus connected field</Button>
          <Button onClick={handleNormalClick} ml={20}>Click to focus normal field</Button>
        </form>
      )} />
    )

}}

</Playground>
