## Refs

You can get and pass `ref` to access the underlying DOM element

<Playground>
  {() => {
    const buttonRef = useRef(null)
    const inputRef = useRef(null)

    handleClick = e => {
      inputRef.current.focus()
    }

    return (
      <Form
        onSubmit={() => {}}
        render={({ handleSubmit }) => (
        <form onSubmit={handleSubmit}>
          <ConnectedField name="firstName" component={InputText} label="First name" ref={inputRef} mb={20} />
          <Button ref={buttonRef} onClick={handleClick}>Click to focus field</Button>
        </form>
      )} />
    )

}}

</Playground>
