# Growl

Customize with `<GrowlTitle>`, `<GrowlAction>` or `<GrowlClose>`.

## Example

To use `Growl` components you'll want toâ€¦

1. Import and spread the `useGrowl` hook
2. Add a `GrowlContainer` to your app
3. Add a trigger to add `Growl` items to the `GrowlContainer`

To test the placement of `Growl` items, open the editor and change the `position` prop of the `GrowlContainer`. 

<Playground>
  {() => {
    // Import and spread the `useGrowl` hook
    const { addGrowl, removeGrowl, content } = useGrowl() 

    // Create a `Growl` item, passing `removeGrowl` through as `onClose`
    const growl = (
      <Growl onClose={removeGrowl}> 
        <GrowlTitle>
          <Icon name="megaphone" size="sm" /> Lorem ipsum dolor sit amet
        </GrowlTitle>
        Class aptent taciti sociosqu ad litora torquent per conubia nostra, per inceptos himenaeos
        <GrowlAction>
          <Button size="sm" variant="tertiary" onClick={console.debug}>
            Action button
          </Button>
        </GrowlAction>
      </Growl>
    )

    // Create a trigger (here we use a button)
    return (
      <>
        <GrowlContainer content={content} position='top-left' />
        <Button onClick={() => addGrowl(growl)}>Show Growl</Button>
      </>
    )

}}

</Playground>
