# Text

The `Text` component allows you to remain consistent with your text sizing and line-height. To add additional padding/margins use the `space` values from (https://styled-system.com/api#space).

## Truncation

Set the number of lines you want to display with `lines`. Your text will be displayed truncated with an ellipsis (`...`) at the end (if necessary).
