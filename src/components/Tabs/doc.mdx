# Tabs

Tabs make it easy to explore and switch between different views.

## Default

`<Tabs>` and `<TabsItem>` are you friends.

<Playground>
   {() => {
    const [bloc, setBloc] = useState()

    return (
      <>
        <Tabs onChangeTab={setBloc}>
          <TabsItem name="tab1">
            <div>Tab1</div>
          </TabsItem>
          <TabsItem name="tab2">
            <div>Tab2</div>
          </TabsItem>
          <TabsItem name="tab3">
            <div>Tab3</div>
          </TabsItem>
          <TabsItem name="tabRandom">
            <div>Long title</div>
          </TabsItem>
          <TabsItem name="link">
            <a href="/#/components/tabs">Link</a>
          </TabsItem>
        </Tabs>
        <div style={{ marginTop: "30px", fontSize: "0.85rem" }}>
          {bloc ? `you have selected ${bloc}` : 'by default, the first child is active'}
        </div>
      </>
    )

}}

</Playground>

## With default active

You can add a default active tab for the first render with `defaultActiveTab` property.

<Playground>
  <Tabs
    defaultActiveTab="tab3"
    onChangeTab={name => console.log(`you have selected tabs "${name}"`)}
  >
    <TabsItem name="tab1">
      <div>Tab1</div>
    </TabsItem>
    <TabsItem name="tab2">
      <div>Tab2</div>
    </TabsItem>
    <TabsItem name="tab3">
      <div>Tab3</div>
    </TabsItem>
  </Tabs>
</Playground>

## With a badge

Add a Badge (or other component) on TabsItem component.

<Playground>
  <Tabs onChangeTab={name => console.log(`you have selected tabs "${name}"`)}>
    <TabsItem name="tab1">
      <div>Tab1</div>
    </TabsItem>
    <TabsItem name="tab2">
      <div>Tab2</div>
      <Badge>4</Badge>
    </TabsItem>
    <TabsItem name="tab3">
      <div>Tab3</div>
      <Badge>new</Badge>
    </TabsItem>
  </Tabs>
</Playground>
