# Tabs

Tabs make it easy to explore and switch between different views.<br />Powered by [Reakit](https://reakit.io/docs/tab/) for accessibility.

## Default

`<Tabs>` `<TabsItem>` and `<TabsPanel>` are you friends.

<Playground>
  <Tabs
    name="tabDemo"
    items={
      <>
        <TabsItem>Tab1</TabsItem>
        <TabsItem>Tab2</TabsItem>
        <TabsItem disabled>Tab3</TabsItem>
      </>
    }
  >
    <TabsPanel>Panel 1</TabsPanel>
    <TabsPanel>Panel 2</TabsPanel>
    <TabsPanel>Panel 3</TabsPanel>
    <TabsPanel>Panel 4</TabsPanel>
    <TabsPanel>Panel 5</TabsPanel>
  </Tabs>
</Playground>

## With default active

You can add a default active tab for the first render with `defaultActiveTab` property.

<Playground>
  <Tabs
    name="tabDemo"
    defaultActiveTab={2}
    items={
      <>
        <TabsItem>Tab1</TabsItem>
        <TabsItem>Tab2</TabsItem>
        <TabsItem>Tab3</TabsItem>
      </>
    }
  >
    <TabsPanel>Panel 1</TabsPanel>
    <TabsPanel>Panel 2</TabsPanel>
    <TabsPanel>Panel 3</TabsPanel>
  </Tabs>
</Playground>

## With a badge

Add a Badge (or other component) on TabsItem component.

<Playground>
  <Tabs
    name="tabDemo"
    defaultActiveTab={2}
    items={
      <>
        <TabsItem>Tab1</TabsItem>
        <TabsItem>
          <span>Tab2 </span>
          <Badge>new</Badge>
        </TabsItem>
        <TabsItem>
          <span>Tab3 </span>
          <Badge>new</Badge>
        </TabsItem>
      </>
    }
  >
    <TabsPanel>Panel 1</TabsPanel>
    <TabsPanel>Panel 2</TabsPanel>
    <TabsPanel>Panel 3</TabsPanel>
  </Tabs>
</Playground>
