# Field

We recommend using [Final Form](https://github.com/final-form/react-final-form), [Redux Form](https://github.com/erikras/redux-form) or [Formik](https://github.com/jaredpalmer/formik) with these components. You can see [usage with these libraries here](/usage-final-form).

Alternatively, you can create your own system to manage the state of your form fields (`value`, `checked` etc).

You can find additional props for each field component in [Fields](/fields/file-upload).

## File Upload

<Playground>
  <Field
    name="random-fifteen"
    component={FileUpload}
    label="File upload example"
    onChange={() => {}}
  >
    {({ fileUrl, isDefault, isHoverAccept, isHoverReject, openFile }) => {
      if (isHoverAccept) {
        return <h3>Drop file!</h3>
      } else if (isHoverReject) {
        return <h3>Can't accept this file</h3>
      } else if (fileUrl) {
        return <img style={{ maxWidth: '100%', maxHeight: '100%' }} src={fileUrl} />
      }
      return <h3>Drop file to upload</h3>
    }}
  </Field>
</Playground>
