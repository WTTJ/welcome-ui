# Field

We recommend using [React Final Form]() or [Redux Form]() with these components. You can see [usage with these libraries here](#usage).

Alternatively, you can create your own form component to manage the state of your form fields (`value`, `checked` etc).

## File Upload

<Playground>
  <Field
    name='random-fifteen'
    fieldType='fileupload'
    label='File upload example'
    onChange={() => {}}
  >
    {({ fileUrl, isDefault, isHoverAccept, isHoverReject, openFile }) => {
      if (isHoverAccept) {
        return <h3>Drop file!</h3>
      } else if (isHoverReject) {
        return <h3>Can't accept this file</h3>
      } else if (fileUrl) {
        return <img style={{ maxWidth: '100%', maxHeight: '100%' }} src={fileUrl} />
      }
      return <h3>Drop file to upload</h3>
    }}
  </Field>
</Playground>
