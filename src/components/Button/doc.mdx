Use size property with `sm` `md`(default) or `lg`.

  <Button size="sm">Small</Button>
  <Button>Medium</Button>
  <Button size="lg">Large</Button>
</Playground>

## Transform to a link

You can transform your Button component with `as` property to add you linker, `href` or `to`... all that you want.

<Playground flex>
  <IconButton
    as="a"
    href="http://www.welcometothejungle.co"
    target="_blank"
    rel="noopener nofollow"
  >
    <Icon name="external_link" size="sm" />
  </IconButton>
  <Button
    as="a"
    href="http://www.welcometothejungle.co"
    target="_blank"
    rel="noopener nofollow"
    variant="secondary"
  >
    <Icon name="sun" size="sm" />
    <span>Welcome</span>
  </Button>
  <Button as={Link} to="/" variant="tertiary">
    Homepage
  </Button>
</Playground>

## Width

You can add `width` property from [styled-system](https://styled-system.com/api#width).

<Playground>
  <Button width={1}>full width</Button>
  <Button width={1 / 2}>half width</Button>
</Playground>

## Radius

You can add `borderRadius` property from [styled-system](https://styled-system.com/api#borders).

<Playground>
  <Button>default</Button>
  <Button borderRadius="10px">10px</Button>
</Playground>
