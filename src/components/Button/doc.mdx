## Formats

Use size property with `xs` `sm` `md`(default) or `lg`.

<Playground>
  <Button size="xs">Tiny</Button>
  <Button size="sm">Small</Button>
  <Button>Medium</Button>
  <Button size="lg">Large</Button>
</Playground>

## Disabled

All disabled buttons have the same style.

<Playground>
  <Button variant="primary-danger" disabled>
    Disabled
  </Button>
</Playground>

## Transform to a link

You can transform your Button component with `as` property to add you linker, `href` or `to`... all that you want.

<Playground flex>
  <Button as="a" href="http://www.welcometothejungle.co" target="_blank" shape="square">
    <Icon name="external_link" />
  </Button>
  <Button
    as="a"
    href="http://www.welcometothejungle.co"
    target="_blank"
    rel="noopener nofollow"
    variant="secondary"
  >
    <Icon name="sun" />
    <span>Welcome</span>
  </Button>
  <Button as={Link} to="/" variant="tertiary">
    Homepage
  </Button>
</Playground>

## Width

You can add `width` property from [xstyled](https://www.smooth-code.com/open-source/xstyled/docs/system-props/#layout).

<Playground>
  <Button width={1}>full width</Button>
  <Button width={1 / 2}>half width</Button>
</Playground>

## Radius

You can add `borderRadius` property from [xstyled](https://www.smooth-code.com/open-source/xstyled/docs/system-props/#borders).

<Playground>
  <Button>default</Button>
  <Button borderRadius="10px">10px</Button>
</Playground>
