## Sizes

Use size property with `sm` `md`(default) or `lg`.

<Playground>
  <Button size="sm">Small</Button>
  <Button>Medium</Button>
  <Button size="lg">Large</Button>
</Playground>

## Disabled

All disabled buttons have the style.

<Playground>
  <Button variant="primary-danger" disabled>
    Disabled
  </Button>
</Playground>

## Width

You can add `width` property from [styled-system](https://styled-system.com/api#width).

<Playground>
  <Button width={1}>full width</Button>
  <Button width={1 / 2}>half width</Button>
</Playground>

## Radius

You can add `borderRadius` property from [styled-system](https://styled-system.com/api#borders).

<Playground>
  <Button>default</Button>
  <Button borderRadius="10px">10px</Button>
</Playground>
