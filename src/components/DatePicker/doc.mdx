# DatePicker

It is based on the [react-datepicker](https://github.com/Hacker0x01/react-datepicker) library.

_Note_: Pass a value of `null` if you don't want the default value of `Date.now()`.

## Simple use case

<Playground>
  <DatePicker onChange={date => console.log('date: ', date)} value={Date.now()} />
</Playground>

## Pass props from [the original library](https://github.com/Hacker0x01/react-datepicker).

<Playground>
  <DatePicker dateFormat="MMM dd YYYY" showMonthDropdown showYearDropdown />
</Playground>

## Give an Icon

<Playground>
  <DatePicker
    icon={<Icon name="date" size="sm" color="nude.500" />}
    onChange={date => console.log('date: ', date)}
    value={Date.now()}
  />
</Playground>
