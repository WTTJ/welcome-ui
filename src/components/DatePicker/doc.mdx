# DatePicker

It is based on the [react-datepicker](https://github.com/Hacker0x01/react-datepicker) library.

_Note_: Pass a value of `null` if you don't want the default value of `Date.now()`.

## Simple use case

<Playground>
  <Form initialValues={{ welcome: Date.now() }}>
    <ConnectedField name="welcome" component={DatePicker} />
  </Form>
</Playground>

## Pass props from [the original library](https://github.com/Hacker0x01/react-datepicker).

<Playground>
  <Form initialValues={{ welcome: Date.now() }}>
    <ConnectedField
      name="welcome"
      component={DatePicker}
      dateFormat="MMM dd yyyy"
      maxDate={new Date()}
      showMonthDropdown
      showYearDropdown
      dropdownMode="scroll"
      yearDropdownItemNumber={30}
    />
  </Form>
</Playground>
