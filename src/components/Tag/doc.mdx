## One character style

When we have only one character, we apply same size for the width and height (it implicitely sets `shape` prop to `square`)

<Playground>
  <Tag size="sm">1</Tag>
  <Tag size="md">1</Tag>
  <Tag size="lg">1</Tag>
</Playground>

## Shape

Set `shape` prop to `square` to get a square no matter the content.

<Playground>
  <Tag size="lg" shape="square">
    1
  </Tag>
  <Tag size="lg" shape="square" width={50}>
    Long text
  </Tag>
</Playground>

Set `shape` prop to `circle` to set a border-radius `50%`.

<Playground>
  <Tag size="lg" shape="circle">
    1
  </Tag>
</Playground>
