# Pagination

## Example

`onChange` return page selected.

<Playground>
  {() => {
    const [selected, setSelected] = useState(1)

    return (
      <>
       <Pagination onChange={setSelected} pageCount={10} />
       <Box mt="20px">Page {selected} is selected !</Box>
      </>
    )

}}

</Playground>

## Range display

When your range display is above your pageCount, all the pages are shown.<br />Change range with `rangeDisplay`, by default is 5.

<Playground>
  {() => {
    const [selected, setSelected] = useState(1)

    return (
      <>
       <Pagination onChange={setSelected} pageCount={5} rangeDisplay={10} />
       <Box mt="20px">Page <strong>{selected} </strong> is selected !</Box>
      </>
    )

}}

</Playground>

## Default page

Use `defaultPage` property.

<Playground>
  {() => {
    const defaultPage = 4
    const [selected, setSelected] = useState(defaultPage)

    return (
      <>
       <Pagination onChange={setSelected} pageCount={10} defaultPage={defaultPage} />
       <Box mt="20px">Page {selected} is selected !</Box>
      </>
    )

}}

</Playground>
