# TimePicker

It is based on the [react-datepicker](https://github.com/Hacker0x01/react-datepicker) library.

_Note_: Pass a value of `null` if you don't want the default value of `Date.now()`.

## Simple use case

<Playground>
  <Form initialValues={{ welcome: Date.now() }}>
    <ConnectedField
      name="welcome"
      component={TimePicker}
      onChange={date => console.log('date: ', date)}
      value={Date.now()}
    />
  </Form>
</Playground>

## Add an icon on the right

<Playground>
  <Form initialValues={{ welcome: Date.now() }}>
    <ConnectedField
      name="welcome"
      component={TimePicker}
      icon={<Icon name="date" color="primary.500" />}
      iconPlacement="right"
    />
  </Form>
</Playground>

## Pass props to TimePicker

<Playground>
  <Form initialValues={{ welcome: Date.now() }}>
    <ConnectedField name="welcome" component={TimePicker} timeIntervals={30} />
  </Form>
</Playground>
