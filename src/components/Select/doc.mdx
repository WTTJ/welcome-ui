# Select

It is based on the [downshift](https://github.com/downshift-js/downshift) library.

## Default

<Playground>
  {() => {
    const ITEMS = [
      { value: 'angular', label: 'Angular' },
      { value: 'backbone', label: 'Backbone' },
      { value: 'elm', label: 'Elm' },
      { value: 'ember', label: 'Ember' },
      { value: 'react', label: 'React' },
      { value: 'vue', label: 'Vue' }
    ]
    const [value, setValue] = useState()
    const handleChange = value => setValue(value)

  }}
</Playground>

## Multiple values

Just add the `isMultiple` prop. Note: to be able to choose multiple values you must pass an array for the `value`.

<Playground>
  {() => {
    const ITEMS = [
      { value: 'angular', label: 'Angular' },
      { value: 'backbone', label: 'Backbone' },
      { value: 'elm', label: 'Elm' },
      { value: 'ember', label: 'Ember' },
      { value: 'react', label: 'React' },
      { value: 'vue', label: 'Vue' }
    ]
    // you can pass either the label, the value or the entire object
    const [value, setValue] = useState(['React', 'ember', ITEMS[2]])
    return (
      <Select isMultiple options={ITEMS} name="multiple" onChange={console.debug} value={value} />
    )
  }}
</Playground>

Passing a `renderMultiple` function allows you to format the selected items below the select.

<Playground>
  {() => {
    const ITEMS = [
      { value: 'behance', label: 'Behance' },
      { value: 'dribbble', label: 'Dribbble' },
      { value: 'facebook', label: 'Facebook' },
      { value: 'github', label: 'Github' },
      { value: 'instagram', label: 'Instagram' },
      { value: 'linkedin', label: 'Linkedin' },
      { value: 'stackoverflow', label: 'Stack Overflow' },
      { value: 'twitter', label: 'Twitter' },
      { value: 'xing', label: 'Xing' },
      { value: 'youtube', label: 'Youtube' }
    ]
    const [value, setValue] = useState(['facebook', 'linkedin'])
    const handleChange = value => setValue(value)
    return (
      <Select
        isMultiple
        name="rich"
        onChange={handleChange}
        options={ITEMS}
        renderMultiple={(selected, handleRemove) => {
          return selected.map(option => (
            <Tag
              key={option.value}
              variant="primary"
              onRemove={() => handleRemove(option.value)}
              mr="sm"
              mt="sm"
            >
              <Icon name={option.value} mr="sm" />
              <Text variant="body3" m="0">
                {option.label}
              </Text>
            </Tag>
          ))
        }}
        value={value}
      />
    )
  }}
</Playground>

## Formatting options

Passing a `renderItem` function allows you to format the options in the list.

Note: if you use `renderItem` with `isSearchable` or `isCreatable` the selected item will _not_ be formatted

Note: if you want to format the options _and_ the placeholder, create a dummy value as the first of your options with an empty string as the value (see code below).

<Playground>

    const ITEMS = [
      { value: '', label: 'Choose a social networkâ€¦' },
      { value: 'behance', label: 'Behance' },
      { value: 'dribbble', label: 'Dribbble' },
      { value: 'facebook', label: 'Facebook' },
      { value: 'github', label: 'Github' },
      { value: 'instagram', label: 'Instagram' },

      { value: 'twitter', label: 'Twitter' },
      { value: 'xing', label: 'Xing' },
      { value: 'youtube', label: 'Youtube' }
    ]
    const [value, setValue] = useState()
    const handleChange = value => setValue(value)
    return (
      <Select
        name="rich"
        onChange={handleChange}
        options={ITEMS}

          <div style={{ display: 'flex', alignItems: 'center' }}>

## Combining props

You can pass any combination of the props above. For example below, we have a `Select` that allows you to search the options, choose multiple options, and create new options.

<Playground>
  {() => {
    const ITEMS = [
      { value: 'angular', label: 'Angular' },
      { value: 'backbone', label: 'Backbone' },
      { value: 'elm', label: 'Elm' },
      { value: 'ember', label: 'Ember' },
      { value: 'react', label: 'React' },
      { value: 'vue', label: 'Vue' }
    ]
    const [value, setValue] = useState(['ember', 'vue'])
    const handleChange = value => setValue(value)
    return (
      <Select
        icon={<Icon name="add" color="nude.500" />}
        isCreatable
        isMultiple
        isSearchable
        options={ITEMS}
        name="combined"
        onChange={handleChange}
        value={value}
      />
    )
  }}
</Playground>

## Sizes

Use size property with `sm` `md` or `lg`(default).

<Playground>
  {() => {
    const ITEMS = [
      { value: 'angular', label: 'Angular' },
      { value: 'backbone', label: 'Backbone' },
      { value: 'elm', label: 'Elm' },
      { value: 'ember', label: 'Ember' },
      { value: 'react', label: 'React' },
      { value: 'vue', label: 'Vue' }
    ]
    const [values, setValues] = useState({ small: 'elm', medium: 'backbone', large: 'angular' })
    const handleChange = (value, e) => setValues({ ...values, [e.target.name]: value })
    return (
      <>
        <Select
          options={ITEMS}
          name="small"
          onChange={handleChange}
          size="sm"
          value={values.small}
        />
        <Select
          options={ITEMS}
          name="medium"
          onChange={handleChange}
          size="md"
          value={values.medium}
        />
        <Select
          options={ITEMS}
          name="large"
          onChange={handleChange}
          size="lg"
          value={values.large}
        />
      </>
    )
  }}
</Playground>
