# Text

<TagVersion version={version} name={name} />

The `Text` component allows you to remain consistent with your text sizing and line-height. To add additional padding/margins use the `space` values from [xstyled](https://xstyled.dev/docs/style-props/).

## Bring your own tag

You can pass a tag name with `as` to use that tag type with the styling from another tag. For exampleâ€¦

<Playground>
  <Text variant="h1" as="p">
    `p` tag styled as an `H1`
  </Text>
  <Text variant="body3" as="h1">
    `H1` tag styled as a `body3`
  </Text>
</Playground>
