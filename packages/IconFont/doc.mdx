# IconFont

If you want to use a font of icons and HTML tags instead of inline SVG, use IconFont !

<TagVersion version={version} name={name} />

## Use icon font

<Code language="bash" isCopyable>{`yarn add @welcome-ui/icons.font ${Object.keys(peerDependencies).join(' ')}`}</Code>
<Code language="bash" isCopyable>{`import { Icons } from '@welcome-ui/icons.font'`}</Code>
or
<Code language="bash" isCopyable>{`import { IconFont } from '@welcome-ui/icons.font'`}</Code>

## Call Icon from Icons import

<Playground>
  <Icons.Twitter />
  <Icons.Right />
  <Icons.Facebook />
</Playground>

## Call Icon from IconFont import

<Playground>
  <IconFont name="facebook" />
  <IconFont name="left" />
  <IconFont name="twitter" />
</Playground>
