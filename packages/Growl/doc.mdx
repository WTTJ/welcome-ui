# Growl

<TagVersion version={version} name={name} />

Customize with `<GrowlTitle>`, `<GrowlAction>` or `<GrowlClose>`.

## Import

<Code>{`import { Growl, GrowlTitle, GrowlAction, GrowlClose } from '@welcome-ui/growl'`}</Code>

## Example

<Playground>
  <Growl onClose={() => console.log('close')}>
    <GrowlTitle>
      <MegaphoneIcon /> Lorem ipsum dolor sit amet
    </GrowlTitle>
    Class aptent taciti sociosqu ad litora torquent per conubia nostra, per inceptos himenaeos
    <GrowlAction>
      <Button size="sm" variant="tertiary">
        Action button
      </Button>
    </GrowlAction>
  </Growl>
</Playground>

## Variants

Variant properties `info` (default), `error` or `warning` with automatic color.

<Playground>

## Customs

Change close component, don't add icon on title, remove action button...

<Playground>
  <Growl>
    <GrowlTitle>Without icon on title, action button and close button</GrowlTitle>
    Class aptent taciti sociosqu ad litora torquent per conubia nostra, per inceptos himenaeos
  </Growl>
  <Growl
    onClose={() => console.log('close')}
    close={
      <GrowlClose>
        <PositiveIcon />
      </GrowlClose>
    }
  >
    <GrowlTitle>Custom close button with GrowlClose component</GrowlTitle>
    Class aptent taciti sociosqu ad litora torquent per conubia nostra, per inceptos himenaeos
  </Growl>
  <Growl onClose={() => console.log('close')} close={<PositiveIcon size="lg" />}>
    <GrowlTitle>Custom close button</GrowlTitle>
    Class aptent taciti sociosqu ad litora torquent per conubia nostra, per inceptos himenaeos
  </Growl>
</Playground>
