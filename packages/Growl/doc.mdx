# Growl

<TagVersion version={version} name={name} />

Customize with `<Growl.Title>`, `<Growl.Action>` or `<Growl.Close>`.

## Install and import

<Usage component={component} name={name} peerDependencies={peerDependencies} />

## Example

<Playground>
  <Growl onClose={() => console.log('close')}>
    <Growl.Title>
      <MegaphoneIcon /> Lorem ipsum dolor sit amet
    </Growl.Title>
    Class aptent taciti sociosqu ad litora torquent per conubia nostra, per inceptos himenaeos
    <Growl.Action>
      <Button size="sm" variant="tertiary">
        Action button
      </Button>
    </Growl.Action>
  </Growl>
</Playground>

## Variants

Variant properties `info` (default), `error` or `warning` with automatic color.

<Playground>
  <Growl onClose={() => console.log('close')}>
    <Growl.Title variant="error">
      <FlagIcon /> Error growl
    </Growl.Title>
    Class aptent taciti sociosqu ad litora torquent per conubia nostra, per inceptos himenaeos
    <Growl.Action>
      <Button size="sm" variant="secondary-danger">
        Action button
      </Button>
    </Growl.Action>
  </Growl>
  <Growl onClose={() => console.log('close')}>
    <Growl.Title variant="warning">
      <ShieldIcon /> Warning growl
    </Growl.Title>
    Class aptent taciti sociosqu ad litora torquent per conubia nostra, per inceptos himenaeos
    <Growl.Action>

        Action button

    </Growl.Action>
  </Growl>
  <Growl onClose={() => console.log('close')}>
    <Growl.Title>

      </Button>
    </Growl.Action>
  </Growl>
</Playground>

## Customs

Change close component, don't add icon on title, remove action button...

<Playground>
  <Growl>
    <Growl.Title>Without icon on title, action button and close button</Growl.Title>
    Class aptent taciti sociosqu ad litora torquent per conubia nostra, per inceptos himenaeos
  </Growl>
  <Growl
    onClose={() => console.log('close')}
    close={
      <Growl.Close>
        <PositiveIcon />
      </Growl.Close>
    }
  >
    <Growl.Title>Custom close button with Growl.Close component</Growl.Title>
    Class aptent taciti sociosqu ad litora torquent per conubia nostra, per inceptos himenaeos
  </Growl>
  <Growl onClose={() => console.log('close')} close={<PositiveIcon size="lg" />}>
    <Growl.Title>Custom close button</Growl.Title>
    Class aptent taciti sociosqu ad litora torquent per conubia nostra, per inceptos himenaeos
  </Growl>
</Playground>
