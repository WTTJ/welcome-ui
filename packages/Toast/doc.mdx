# Toast

## useToast

Use `useToast` hook to show the toast element. Example below with a custom component.

<Playground>
  {() => {
    const toast = useToast()
    return (
      <Button
        onClick={() =>
          toast(
            <Box
              backgroundColor="light.900"
              borderWidth="1px"
              borderStyle="solid"
              borderColor="nude.300"
              borderRadius="sm"
              padding="sm"
            >
              Lorem ipsum dolor sit amet
            </Box>
          )
        }
      >
        Show Toast
      </Button>
    )
  }}
</Playground>

## Options

You can choose between 2 options :

- **position** : one of `top-left`, `top`, `top-right`, `bottom-left`, `bottom` (default) or `bottom-right`
- **duration** : before closing element, in ms, with default `3000`. Can also be set to `null` if not automatic close your element.

<Playground>
  {() => {
    const toast = useToast()
    return (
      <>
        {['top-left', 'top', 'top-right', 'bottom-left', 'bottom', 'bottom-right'].map(position => (
          <Button
            key={position}
            onClick={() =>
              toast(
                <Box
                  backgroundColor="light.900"
                  borderWidth="1px"
                  borderStyle="solid"
                  borderColor="nude.300"
                  borderRadius="sm"
                  padding="sm"
                >
                  Lorem ipsum dolor sit amet
                </Box>,
                { position, duration: 6000 }
              )
            }
          >
            {position}
          </Button>
        ))}
      </>
    )
}}

</Playground>

## Snackbar

Use `Snackbar` component to show action notification.

<Playground>
  {() => {
    const toast = useToast()
    return (
      <>
        <Button
          onClick={() =>
            toast(
              <Snackbar>
                <CelebrateIcon />
                <span>Lorem ipsum dolor sit amet taciti sociosqu ad</span>
              </Snackbar>
            )
          }
        >
          Info (Default)
        </Button>
        <Button
          onClick={() =>
            toast(
              <Snackbar variant="error">
                <NegativeIcon />
                <span>Lorem ipsum dolor sit amet taciti sociosqu ad</span>
              </Snackbar>
            )
          }
        >
          Error
        </Button>
        <Button
          onClick={() =>
            toast(
              <Snackbar variant="warning">
                <MegaphoneIcon />
                <span>Lorem ipsum dolor sit amet taciti sociosqu ad</span>
              </Snackbar>
            )
          }
        >
          Warning
        </Button>
        <Button
          onClick={() =>
            toast(
              <Snackbar variant="success">
                <CheckIcon />
                <span>Lorem ipsum dolor sit amet taciti sociosqu ad</span>
              </Snackbar>
            )
          }
        >
          Success
        </Button>
      </>
    )
  }}
</Playground>
