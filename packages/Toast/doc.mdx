## useToast

Use `useToast` hook to show the toast element. Example below with a custom component.

<Playground>
  {() => {
    const toast = useToast()
    const Notification = () => (
      <Box
        backgroundColor="light.900"
        borderWidth="1px"
        borderStyle="solid"
        borderColor="nude.300"
        borderRadius="sm"
        padding="sm"
      >
        Lorem ipsum dolor sit amet
      </Box>
    )
    return (
      <Button onClick={() => toast(<Notification />)}>
        Show Toast
      </Button>
    )
}}

</Playground>

## Options

The `toast` function takes a component to be displayed and an `options` object:

- **position** : one of `top-left`, `top`, `top-right`, `bottom-left`, `bottom` (default) or `bottom-right`
- **duration** : before closing element, in ms, with default `5000`. Can also be set to `null` if not automatic close your element.

<Playground>
  {() => {
    const toast = useToast()
    const Notification = () => (
      <Box
        backgroundColor="light.900"
        borderWidth="1px"
        borderStyle="solid"
        borderColor="nude.300"
        borderRadius="sm"
        padding="sm"
      >
        Lorem ipsum dolor sit amet
      </Box>
    )
    return (
      <>
        {['top-left', 'top', 'top-right', 'bottom-left', 'bottom', 'bottom-right'].map(position => (
          <Button
            key={position}
            onClick={() =>
              toast(
                <Notification />,
                { position, duration: 6000 }
              )
            }
          >
            {position}
          </Button>
        ))}
      </>
    )
}}

</Playground>

## Supplied components

You can create your own notification component to display in the Toast or one of our default components: `Snackbar` and `Growl`.

In general :

- `Snackbar` is used to display a temporary, information-only notification, and
- `Growl` is used to display a more important notification with an action (that requires the user to close the notification)

### Snackbar

<Playground>

    const toast = useToast()
    return (

        <Button

              <Toast.Snackbar>

                <span>Lorem ipsum dolor sit amet taciti sociosqu ad</span>
              </Toast.Snackbar>
            )
          }
        >
          Info (Default)
        </Button>
        <Button
          onClick={() =>
            toast(
              <Toast.Snackbar variant="error">
                <NegativeIcon />
                <span>Lorem ipsum dolor sit amet taciti sociosqu ad</span>
              </Toast.Snackbar>
            )
          }
        >
          Error
        </Button>
        <Button
          onClick={() =>
            toast(
              <Toast.Snackbar variant="warning">
                <MegaphoneIcon />
                <span>Lorem ipsum dolor sit amet taciti sociosqu ad</span>
              </Toast.Snackbar>
            )
          }
        >
          Warning
        </Button>
        <Button
          onClick={() =>
            toast(

                <span>Lorem ipsum dolor sit amet taciti sociosqu ad</span>

          Success

      </>

  }}
</Playground>
