# Group

<TagVersion version={version} name={name} />

Children grouped without space or radius.

## Install and import

<Code language="bash">{`
  yarn add @welcome-ui/group ${Object.keys(peerDependencies).join(' ')}
  …
  import { Group } from '@welcome-ui/group'
`}</Code>

## Example

This works only with the `Button` component for now — more examples will be added soon.

### Button

<Playground>
  <Group>
    <Button>First</Button>

    <Button>Third</Button>

  </Group>
</Playground>

## Overload

`Group` overloads the child components properties.

<Playground>
  <Group variant="tertiary" size="xs">
    <Button>First</Button>
    <Button variant="secondary">Second</Button>
    <Button variant="tertiary">Third</Button>
    <Button variant="tertiary">Last</Button>
  </Group>
</Playground>
