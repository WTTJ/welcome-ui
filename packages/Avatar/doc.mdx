## Without image

Without image we add initials on the avatar with array property `words`. We get by default ramdom color from sub colors to create the background. You can override with `color`. Font size is calculated automatically and can be override with `fontSize`property.

<Playground>
  <Avatar alt="Avatar with image" words={['Welcome', 'Jungle']} size="sm" />
  <Avatar alt="Avatar with image" words={['Welcome', 'Jungle']} />
  <Avatar alt="Avatar with image" words={['Welcome', 'Jungle']} size="lg" />
  <Avatar alt="Avatar with image" words={['Welcome', 'Jungle']} size="xl" />
  <Avatar alt="Avatar with image" words={['Welcome', 'Jungle']} size="xxl" />

    fontSize={20}
  />
</Playground>
