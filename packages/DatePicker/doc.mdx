# DatePicker

<TagVersion version={version} name={name} />

It is based on the [react-datepicker](https://github.com/Hacker0x01/react-datepicker) library.

_Note_: Pass a value of `null` if you don't want the default value of `Date.now()`.

## Import

<Code>{`import { DatePicker } from '@welcome-ui/date-picker'`}</Code>

## Simple use case

<Playground>
  <Form initialValues={{ welcome: Date.now() }}>
    <ConnectedField component={DatePicker} label="Date" name="welcome" />
  </Form>
</Playground>

## Pass props from the original library.

<Playground>
  <Form initialValues={{ welcome2: Date.now() }}>
    <ConnectedField
      component={DatePicker}
      dateFormat="MMM dd yyyy"
      label="Date"
      maxDate={new Date()}

  </Form>
</Playground>
