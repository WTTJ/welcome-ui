## Separator

You can customize your separator.

<Playground>
  <Breadcrumb separator="/">
    <Breadcrumb.Item href="/">Introduction</Breadcrumb.Item>
    <Breadcrumb.Item href="/">Components</Breadcrumb.Item>
    <Breadcrumb.Item>Breadcrumb</Breadcrumb.Item>
  </Breadcrumb>

  <Breadcrumb separator="-">
    <Breadcrumb.Item href="/">Introduction</Breadcrumb.Item>
    <Breadcrumb.Item href="/">Components</Breadcrumb.Item>
    <Breadcrumb.Item>Breadcrumb</Breadcrumb.Item>
  </Breadcrumb>
</Playground>

## Add global Link component

Add a global link component for the items with `renderChildrenAs`. Example with an `import { Link } from '@welcome-ui/link'`.

<Playground>
  <Breadcrumb renderChildrenAs={Link}>
    <Breadcrumb.Item href="/">Introduction</Breadcrumb.Item>
    <Breadcrumb.Item href="/">Components</Breadcrumb.Item>
    <Breadcrumb.Item>Breadcrumb</Breadcrumb.Item>
  </Breadcrumb>
</Playground>
