### Items without link will be disable

We are looking for `href` prop. (or **to** prop for react-router)

<div data-playground="child-disabled.tsx" data-component="Breadcrumb"></div>

### Usage with React Router

```jsx
import { Link as RouterLink } from 'react-router-dom'

return (
  <Breadcrumb>
    <Breadcrumb.Item as={RouterLink} to="/">
      Introduction
    </Breadcrumb.Item>
    <Breadcrumb.Item as={RouterLink} to="/components">
      Components
    </Breadcrumb.Item>
    <Breadcrumb.Item as={RouterLink} to="/components/breadcrumb">
      Breadcrumb
    </Breadcrumb.Item>
  </Breadcrumb>
)
```
