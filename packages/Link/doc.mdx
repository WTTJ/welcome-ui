# Link

<TagVersion version={version} name={name} />

A link component who can get `as` property to pass custom linker.

This component is a child from `UniversalLink` component.

## Install and import

<Usage component={component} name={name} peerDependencies={peerDependencies} />

## Variants

Can be `primary` by default or `secondary`

<Playground>
  <Box>
    <Link href="#">primary link</Link>
  </Box>
  <Box>
    <Link href="#" variant="secondary">
      secondary link
    </Link>
  </Box>
</Playground>

It can also be `disabled`

<Playground>
  <Link disabled href="#">
    primary link
  </Link>
</Playground>

## It works on multiple lines too

<Playground>
  <Box>
    <Link as={LinkDocz} display="inline" to="/components/shape" variant="primary">
      Ipsum ipsam fugiat doloribus sit hic ducimus dolorem Aperiam unde adipisci deserunt corporis
      neque commodi. Quasi odit esse molestias reprehenderit incidunt Velit non expedita atque
      exercitationem enim! Quidem quasi nihil.
    </Link>
  </Box>
</Playground>

## What is underlined by default?

`Link` will add an underline to text nodes & `Text` components

<Playground>
  <Box>
    <Link as={LinkDocz} to="/components/shape">
      <Shape height="30px" shape="circle" width="30px">
        <img src="https://avatars3.githubusercontent.com/u/13100706?s=200&v=4" />
      </Shape>
      This is a text node
    </Link>
  </Box>
  <Box>
    <Link as={LinkDocz} to="/components/shape">
      <Shape height="30px" shape="circle" width="30px">
        <img src="https://avatars3.githubusercontent.com/u/13100706?s=200&v=4" />
      </Shape>
      <Text>This is a text in a Text component</Text>
    </Link>
  </Box>
</Playground>

## Security

For a target `_blank` we add `rel='noopener noreferrer'`

<Playground>
  <Link href="#" target="_blank">
    with a target blank
  </Link>
</Playground>

## Example with React Router

<Code>
  {`import { Link as RouterLink } from 'react-router-dom'

<Link as={RouterLink} to="/jobs">
  Jobs
</Link>`}
</Code>
