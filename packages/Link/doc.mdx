# Link

<TagVersion version={version} name={name} />

A link component who can get `as` property to pass custom linker.

This component is a child from `UniversalLink` component.

## Install and import

<Usage component={component} name={name} peerDependencies={peerDependencies} />

## Variants

Can be `primary` by default or `secondary`

<Playground>
  <Box>
    <Link href="#">primary link</Link>
  </Box>
  <Box>
    <Link href="#" variant="secondary">
      secondary link
    </Link>
  </Box>
</Playground>

It can also be `disabled`

<Playground>
  <Link disabled href="#">
    primary link
  </Link>
</Playground>

## OnClick

It's not required to be an `<a>`, it can be a span with an onClick

  <Link as="span" onClick={() => alert('itâ€˜s a span!')}>

  </Link>
  <Link as="span" disabled onClick={() => alert('itâ€˜s a span!')}>
    Disabled
  </Link>
</Playground>

## Special case

Use variant to `primary-underline-span` to add undeline primary link style on span

<Playground>
  <Box>
    <Link as={LinkDocz} to="/components/shape" variant="primary-underline-span">
      <Shape height="30px" shape="circle" width="30px">
        <img src="https://avatars3.githubusercontent.com/u/13100706?s=200&v=4" />
      </Shape>
      <span>see shape component</span>
    </Link>
  </Box>
</Playground>

## Security

For a target `_blank` we add `rel='noopener noreferrer'`

<Playground>
  <Link href="#" target="_blank">
    with a target blank
  </Link>
</Playground>
