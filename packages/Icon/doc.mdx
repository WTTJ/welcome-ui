# Icon

## Formats

Use size property with `xs`, `sm`, `md`, `lg` or `xl`.

<Playground>
  <TwitterIcon size="xs" />
  <TwitterIcon size="sm" />
  <TwitterIcon />
  <TwitterIcon size="lg" />
  <TwitterIcon size="xl" />
</Playground>

## Colour

Icons inherit the colour from their parent

<Playground>
  <TwitterIcon color="blue" size="xl" />
  <TwitterIcon color="orange" size="xl" />
  <TwitterIcon color="green" size="xl" />
</Playground>
