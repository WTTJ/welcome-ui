# Icon

## Formats

Use size property with `xs`, `sm`, `md`, `lg` or `xl`.

<Playground>
  <Icon name="twitter" size="xs" />
  <Icon name="twitter" size="sm" />
  <Icon name="twitter" />
  <Icon name="twitter" size="lg" />
  <Icon name="twitter" size="xl" />
</Playground>

## Colour

Icons inherit the colour from their parent

<Playground>
  <Icon color="blue" name="twitter" size="xl" />
  <Icon color="orange" name="twitter" size="xl" />
  <Icon color="green" name="twitter" size="xl" />
</Playground>
