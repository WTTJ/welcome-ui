## Install and import individual icons

<Code language="bash" isCopyable>{`yarn add @welcome-ui/icons.avatar ${Object.keys(peerDependencies).join(' ')}`}</Code>
<Code language="bash" isCopyable>{`import { AvatarIcon } from '@welcome-ui/icons.avatar'`}</Code>

## Orâ€¦ install and import all icons

<Code language="bash" isCopyable>{`yarn add @welcome-ui/icons ${Object.keys(peerDependencies).join(' ')}`}</Code>
<Code language="bash" isCopyable>{`import * as Icons from '@welcome-ui/icons'`}</Code>

## Size

Use size property with `xs`, `sm`, `md`, `lg` or `xl`.

<Playground>
  <TwitterIcon size="xs" />
  <TwitterIcon size="sm" />
  <TwitterIcon />
  <TwitterIcon size="lg" />
  <TwitterIcon size="xl" />
</Playground>

## Colour

Icons inherit the colour from their parent

<Playground>
  <TwitterIcon color="blue" name="twitter" size="xl" />
  <TwitterIcon color="orange" name="twitter" size="xl" />
  <TwitterIcon color="green" name="twitter" size="xl" />
</Playground>
