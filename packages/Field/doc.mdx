# Field

<TagVersion version={version} name={name} />

We recommend using [Final Form](https://github.com/final-form/react-final-form), [Redux Form](https://github.com/erikras/redux-form) or [Formik](https://github.com/jaredpalmer/formik) with these components. If so, use `ConnectedField` for your fields (as in the examples below).

The examples below give you an overview of common props (e.g. `size`, `disable`, `required` etc.)

You can find props for each field component in the other `Fields` pages.
