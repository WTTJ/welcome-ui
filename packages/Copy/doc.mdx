# useCopyText

<TagVersion version={version} name={name} />

A simple hook to copy on your clipboard with copyToClipboard library.

## Install and import

<Usage component={component} name={name} peerDependencies={peerDependencies} />

## Example

Copy the window location.

<Playground>
  {() => {
    const copyRef = React.useRef()
    const [copy, copied] = useCopyText(copyRef)
    return (
      <>
        <div ref={copyRef}>Fishcakes!</div>
        <Button onClick={copy}>
          {copied ? (
            <>
              <CheckIcon />
              <span>Copied!</span>
            </>
          ) : (
            <>
              <LinkIcon />
              <span>Copy link</span>
            </>
          )}
        </Button>
      </>
    )
  }}
</Playground>

## Remove timeout

Add a second parameter to remove the copied element.

<Playground>
  {() => {
    const [copy, copied] = useCopyText('<Button>Copied source</Button>', 2000)
    return (
      <>
        <Button variant="primary-warning" onClick={copy}>
          <SourceIcon />
          <span>Copy</span>
        </Button>
        {copied && (
          <Box mt="sm">
            <CheckIcon color="success.500" /> Copied!
          </Box>
        )}
      </>
    )
  }}
</Playground>
