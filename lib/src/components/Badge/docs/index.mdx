### Sizes

Use size property with `sm`, `md` (default), or `lg`. For status indicators or notification information you can use `sm` size.

<div data-playground="size.tsx" data-component="Badge"></div>

### Variants

Use variant property with `brand`, `blue`, `neutral`, or `warm` (default)

<div data-playground="variant.tsx" data-component="Badge"></div>

### withNumberAbbreviation

If a number is higher than 99, we replace this number by 99+, simply add property `withNumberAbbreviation`. _The children need to be a number!_
