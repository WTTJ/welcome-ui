## useTab

We use `useTab` from [Ariakit Tab](https://ariakit.org/reference/use-tab-store) for the state of the Tab.

Pass options to `useTab`:

- **defaultSelectedId**: e.g. `const store = useTab({ defaultSelectedId: 'tab2' })`
