### Image

Pass an image url with `src="you-url"`.

<div data-playground="images.tsx" data-component="Avatar"></div>

### Placeholder

If no image is provided and no name is given, we display a generic placeholder.

<div data-playground="placeholder.tsx" data-component="Avatar"></div>

### Text

If no image is provided we create a two-letter initial based on the name. We choose a random colour from secondaries theme colours for the background. For `xs`and `sm` sizes, we only display the first letter.

<div data-playground="texts.tsx" data-component="Avatar"></div>

### Size

You can use `xs`, `sm`, `md` and `lg` sizes.

<div data-playground="sizes.tsx" data-component="Avatar"></div>
