## Bring your own tag

You can pass a tag name with `as` to use that tag type with the styling from another tag. For exampleâ€¦

```jsx
<Text variant="h3" as="p" mt="0" mb="xl">
  p tag styled as an H3
</Text>
<Text variant="sm" as="h1">
  H1 tag styled as a sm
</Text>
```
