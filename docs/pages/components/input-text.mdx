## Usage

```jsx
function () {
  const [value, setValue] = React.useState('')

  const handleChange = (event) => {
    setValue(event.target.value)
  }

  return <InputText name="firstName" placeholder="Boaty" value={value} onChange={handleChange} />
}
```

## Label

Use `label` and Field component to add an Label to your field

```jsx
function () {
  const [value, setValue] = React.useState('')

  const handleChange = (event) => {
    setValue(event.target.value)
  }

  return (
    <Field label="First name">
      <InputText placeholder="Boaty" value={value} onChange={handleChange} />
    </Field>
  )
}
```

## Hint

Use `hint` and Field component to add an Hint to your field

```jsx
function () {
  const [value, setValue] = React.useState('')

    setValue(event.target.value)

  return (
    <Field hint="a hint">

```

## Required

Use `required` on Field component to add an asterix on required field

```jsx
function () {
  const [value, setValue] = React.useState('')

  const handleChange = (event) => {
    setValue(event.target.value)
  }

  return (
    <Field label="First name" required>
      <InputText placeholder="Boaty" value={value} onChange={handleChange} />
    </Field>
  )
}
```

## Clearable

Use the `isClearable` prop to add a cross button to remove the content (value) of the `TextInput`

```jsx
function () {
  const [value, setValue] = React.useState('')

  const handleChange = (event) => {
    setValue(event.target.value)
  }

  return <InputText name="firstName2" placeholder="Boaty" value={value} onChange={handleChange} isClearable />
}
```

## Icon

Pass an `icon` through to decorate your `InputText`.

Use `iconPlacement` to put on left (default) or right.

```jsx
function () {
  const [value, setValue] = React.useState('')

  const handleChange = (event) => {
    setValue(event.target.value)
  }

  return (
    <Stack spacing="xl">
      <InputText icon={<AvatarIcon />} name="firstName3" placeholder="Boaty" value={value} onChange={handleChange} />
      <InputText icon={<AvatarIcon />} iconPlacement="right" name="firstName3" placeholder="Boaty" value={value} onChange={handleChange} />
    </Stack>
  )
}
```

## Sizes

Use size propety with option:
- `xs` (24px)
- `sm` (32px)
- `md` (40px - default)
- `lg` (48px)

```jsx
<Stack spacing="xl">
  <InputText icon={<AvatarIcon />} name="extra-small" placeholder="Placeholder" size="xs" />
  <InputText icon={<AvatarIcon />} name="small" placeholder="Placeholder" size="sm" />
  <InputText icon={<AvatarIcon />} name="medium" placeholder="Placeholder" />
  <InputText icon={<AvatarIcon />} name="large" placeholder="Placeholder" size="lg" />
</Stack>
```

## Variants

Use `warning`, `error`, `info` or `success` properties to add a variant status on your fields. The label, hint or border color are set by magic ðŸª„

```jsx
<Field label="Label" warning="A warning">
  <InputText placeholder="Placeholder" />
</Field>
<Field label="Label" error="An error">
  <InputText placeholder="Placeholder" />
</Field>
<Field label="Label" info="An info">
  <InputText placeholder="Placeholder" />
</Field>
<Field label="Label" success="A success">
  <InputText placeholder="Placeholder" />
</Field>
```

## Disabled

Property `disabled` can be pass to the component or Field component

```jsx
<InputText disabled name="firstName5" placeholder="Boaty" />

<Field disabled label="First name" required>
  <InputText name="firstName6" placeholder="Boaty" />
</Field>
```

## Transparent

Pass `transparent` to remove background-color and border-color

```jsx
  <InputText icon={<AvatarIcon />} transparent name="firstName4" placeholder="Boaty" />
```

## Inline

An example to put inline the input text with a label

```jsx
<Field flexDirection="row" label="First name" required>
  <InputText name="firstName4" placeholder="Boaty" />
</Field>
```
