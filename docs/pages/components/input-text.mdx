## Default

```jsx
<Form initialValues={{}}>
  <ConnectedField
    name="firstName"
    component={InputText}
    placeholder="Boaty"
    label="First name"
    required
  />
</Form>
```

## Clearable

Use the `isClearable` prop to add a cross button to remove the content (value) of the `TextInput`

```jsx
<Form initialValues={{ firstName2: 'Trevor' }}>
  <ConnectedField
    component={InputText}
    isClearable
    label="First name"
    name="firstName2"
    placeholder="Boaty"
  />
</Form>
```

## Icon

Pass an `icon` through to decorate your `InputText`

```jsx
<Form initialValues={{}}>
  <ConnectedField
    component={InputText}
    icon={<AvatarIcon color="light.100" />}
    label="First name"
    name="firstName3"
    placeholder="Boaty"
  />
</Form>
```
