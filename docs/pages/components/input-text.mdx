## Default

```jsx
<Form initialValues={{}}>
  <Field label="First name" required>
    <InputText name="firstName" placeholder="Boaty" />
  </Field>
</Form>
```

## Clearable

Use the `isClearable` prop to add a cross button to remove the content (value) of the `TextInput`

```jsx
<Form initialValues={{ firstName2: 'Trevor' }}>
  <Field label="First name">
    <InputText isClearable name="firstName2" placeholder="Boaty" />
  </Field>
</Form>
```

## Icon

Pass an `icon` through to decorate your `InputText`

```jsx
<Form initialValues={{}}>
  <Field label="First name">
    <InputText icon={<AvatarIcon color="light.100" />} name="firstName3" placeholder="Boaty" />
  </Field>
</Form>
```

## Sizes

```jsx
    </Field>
    <Field label="I'm a large sized field which is my default size">

```
