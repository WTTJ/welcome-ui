## Usage

Use `usePopoverStore`, `Popover`, `Popover.Content` and `Popover.Trigger` to create a simple Popover.

```jsx
function() {
  const store = usePopoverStore()

  return (
    <>
      <Popover.Trigger as={Button} store={store}>
        Open Popover
      </Popover.Trigger>
      <Popover aria-label="usage popover" store={store}>
        <Popover.Content>
          Praesent sit amet quam ac velit faucibus dapibus. Quisque sapien ligula, rutrum quis
          aliquam nec, convallis sit amet erat. Mauris auctor blandit porta.
        </Popover.Content>
      </Popover>
    </>
  )
}
```

## Title

Use `Popover.Title` props to create a predefined title block.

```jsx
function() {
  const store = usePopoverStore({ placement: 'top' })

  return (
    <>
      <Popover.Trigger as={Button} store={store}>
        Open Popover
      </Popover.Trigger>
      <Popover aria-label="title popover" store={store}>
          <Popover.Title>Amazing title</Popover.Title>
          <Popover.Content>
            Praesent sit amet quam ac velit faucibus dapibus. Quisque sapien ligula, rutrum quis
            aliquam nec, convallis sit amet erat. Mauris auctor blandit porta.
          </Popover.Content>
      </Popover>
    </>
  )
}
```

## Close button

`withCloseButton` show close button.

```jsx
function() {
  const store = usePopoverStore({ withCloseButton: true })

  return (
    <>
      <Popover.Trigger as={Button} store={store}>
        Open Popover
      </Popover.Trigger>
      <Popover aria-label="close button popover" store={store}>
          <Popover.Title>Amazing title</Popover.Title>
          <Popover.Content>
            Praesent sit amet quam ac velit faucibus dapibus.<br />
            Quisque sapien ligula, rutrum quis aliquam nec, convallis sit amet erat.<br />
            Mauris auctor blandit porta.
          </Popover.Content>
      </Popover>
    </>
  )
}
```

## Hover to open

Use `triggerMethod: 'hover'` on `usePopoverStore` to open and close the popover by hovering it.

```jsx
function() {
  const store = usePopoverStore({ triggerMethod: 'hover' })

  return (
    <>
      <Popover.Trigger as={Button} store={store}>
        Open Popover
      </Popover.Trigger>
      <Popover aria-label="hover to open popover" store={store}>
          <Popover.Title>Amazing title</Popover.Title>
          <Popover.Content>
            Praesent sit amet quam ac velit faucibus dapibus.<br />
            Quisque sapien ligula, rutrum quis aliquam nec, convallis sit amet erat.<br />
            Mauris auctor blandit porta.
          </Popover.Content>
      </Popover>
    </>
  )
}
```

## usePopoverStore

We use `usePopoverStore` from [Ariakit Popover](https://ariakit.org/components/popover) for the state of the popover.

Pass options to `usePopoverStore` :

- `defaultOpen`: e.g. `const store = usePopoverStore({ defaultOpen: true })`
- `triggerMethod` : `click` or `hover`
- `withCloseButton` : `bool`, show/hide cross to close popover

When `triggerMethod` is set to hover

- `showTimeout`: `number` by default to `500`, show after x milliseconds on hover the trigger
- `hideTimeout`: `number` by default to `300`, close after x milliseconds on mouse lease popover

And the hook returns (among other things):

- `useState('open')` : whether the popover is currently open
- `hide` : a function to hide the popover
- for the others options, go to [Ariakit Popover](https://ariakit.org/components/popover)
