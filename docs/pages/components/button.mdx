## Variants

### Basics

```jsx row
<Button>Primary</Button>
<Button variant="secondary">Secondary</Button>
<Button variant="tertiary">Tertiary</Button>
<Button variant="ghost">Ghost</Button>
```

### States

```jsx row
<Button variant="primary-success">Primary success</Button>
<Button variant="primary-warning">Primary warning</Button>
<Button variant="primary-danger">Primary danger</Button>
<Button variant="primary-info">Primary info</Button>
<Button variant="secondary-success">Secondary success</Button>
<Button variant="secondary-warning">Secondary warning</Button>
<Button variant="secondary-danger">Secondary danger</Button>
<Button variant="secondary-info">Secondary info</Button>
```

## Formats

Use size property with `xxs`, `xs`, `sm`, `md` (default) or `lg`.

```jsx row
<Button size="xxs">XX-Small</Button>
<Button size="xs">X-Small</Button>
<Button size="sm">Small</Button>
<Button>Medium</Button>
<Button size="lg">Large</Button>
```

## Disabled

All disabled buttons have the same style.

```jsx
<Button variant="primary-danger" disabled>
  Disabled
</Button>
```

## Loading

```jsx
<Button variant="primary" isLoading>
  Loading
</Button>
```

## With icons

```jsx row
<Button>
  <EarthIcon />
  <span>Button</span>
</Button>
<Button variant="secondary">
  <EarthIcon />
  <span>Button</span>
</Button>
<Button variant="tertiary">
  <EarthIcon />
  <span>Button</span>
</Button>
<Button disabled>
  <EarthIcon />
  <span>Button</span>
</Button>
```

## Shape

You can set a `cirle` or `square` shape with `w` & `h` to have buttons perfect for icons.

```jsx row
<Button shape="square" size="xs">
  <EarthIcon />
</Button>
<Button shape="circle" size="xs">
  <EarthIcon />
</Button>
<Button shape="square" size="sm">
  <EarthIcon />
</Button>
<Button shape="circle" size="sm">
  <EarthIcon />
</Button>
<Button shape="square">
  <EarthIcon />
</Button>
<Button shape="circle">
  <EarthIcon />
</Button>
<Button shape="square" size="lg">
  <EarthIcon />
</Button>
<Button shape="circle" size="lg">
  <EarthIcon />
</Button>
```

## Transform to a link

You can transform your Button component with `as` property to add you linker, `href` or `to`... all that you want.

```jsx
<Button as="a" href="https://www.welcometothejungle.com" target="_blank" rel="noopener nofollow">
  <SunIcon />
  <span>Welcome</span>
</Button>
```

## Width

You can add `w` property from [xstyled](https://xstyled.dev/docs/width/).

```jsx
<Button w="100%">full width</Button>
<Button w={1 / 2}>half width</Button>
```
