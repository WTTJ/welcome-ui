```jsx row
<Icons.Twitter />
<Icons.Right />
<Icons.Facebook />
```

## IconFont import

```jsx row
<IconFont name="facebook" />
<IconFont name="left" />
<IconFont name="twitter" />
```

## Css import

To import IconFont css stylesheet, you should use this import path:

<code language="javascript">{`require('@welcome-ui/icons.font/fonts/welcome-icon-font.css')`}</code>

Here is an example on how to use css class to display icons:

```jsx row
<i className="wui-icon-facebook" />
<i className="wui-icon-left" />
<i className="wui-icon-twitter" />
```
