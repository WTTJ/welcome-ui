## Install and import individual icons

<code language="bash">
  {`yarn add @welcome-ui/icons.avatar ${Object.keys(peerDependencies).join(' ')}`}
</code>
<code language="bash">{`import { AvatarIcon } from '@welcome-ui/icons.avatar'`}</code>

## Orâ€¦ install and import all icons

<code language="bash" isCopyable>
  {`yarn add @welcome-ui/icons ${Object.keys(peerDependencies).join(' ')}`}
</code>
<code language="bash" isCopyable>{`import * as Icons from '@welcome-ui/icons'`}</code>

## Size

Use size property with `xs`, `sm`, `md`, `lg` or `xl`.

```jsx row
<TwitterIcon size="xs" />
<TwitterIcon size="sm" />
<TwitterIcon />
<TwitterIcon size="lg" />
<TwitterIcon size="xl" />
```

## Colour

Icons inherit the colour from their parent.

```jsx row
<TwitterIcon color="blue" size="xl" />
<TwitterIcon color="orange" size="xl" />
<TwitterIcon color="green" size="xl" />
```
