## Usage

```jsx
<Form
  initialValues={{}}
  validate={values => (!values.social ? { social: 'You must choose a social network' } : {})}
>
  <ConnectedField
    component={RadioGroup}
    flexDirection="row"
    hint="Most popular is Twitter"
    label="Social networks"
    mb="xl"
    name="social"
    options={constants.ITEMS}
    required
  />
  <Button type="submit">Submit</Button>
</Form>
```
