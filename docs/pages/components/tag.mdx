## Sizes

Use size property with `sm` `md` or `lg`.

```jsx row
<Tag size="sm" variant="secondary">
  Small
</Tag>
<Tag size="md" variant="secondary">
  Medium
</Tag>
<Tag size="lg" variant="secondary">
  Large
</Tag>
```

## One character style

When we have only one character, we apply same size for the width and height (it implicitely sets `shape` prop to `square`)

```jsx row
<Tag size="sm" variant="secondary">
  1
</Tag>
<Tag size="md" variant="secondary">
  1
</Tag>
<Tag size="lg" variant="secondary">
  1
</Tag>
```

## Shape

Set `shape` prop to `square` to get a square no matter the content.

```jsx row
<Tag size="lg" shape="square" variant="secondary">
  1
</Tag>
<Tag size="lg" shape="square" variant="secondary" width={50}>
  Long text
</Tag>
```

Set `shape` prop to `circle` to set a border-radius `50%`.

```jsx
<Tag size="lg" shape="circle" variant="secondary">
  1
</Tag>
```

## OnRemove

Simply add a function `onRemove` to reveal a cross icon with a remove action (or other).

```jsx
function() {
  const [isHide, setHide] = useState(false)
  return isHide ? (
    <Text fontSize="body3" fontWeight="bold" color="danger.500" m={0}>
      Tag removed
    </Text>
  ) : (
    <Tag onRemove={() => setHide(true)}>Example of remove tag</Tag>
  )
}
```
