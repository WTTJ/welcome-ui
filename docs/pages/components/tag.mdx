## Sizes

Use size property with `xs`, `sm`, or `md`.

```jsx row
<Tag size="xs">
  Tiny
</Tag>
<Tag size="sm">
  Small
</Tag>
<Tag size="md">
  Medium
</Tag>
```

## One character style

When we have only one character, we apply same size for the width and height (it implicitely sets `shape` prop to `square`)

```jsx row
<Tag size="xs">
  1
</Tag>
<Tag size="sm">
  1
</Tag>
<Tag size="md">
  1
</Tag>
```

## Shape

Set `shape` prop to `square` to get a square no matter the content.

```jsx row
<Tag size="md" shape="square">
  1
</Tag>
<Tag size="md" shape="square" w={50}>
  Long text
</Tag>
```

Set `shape` prop to `circle` to set a border-radius `50%`.

```jsx
<Tag size="md" shape="circle">
  1
</Tag>
```

## OnRemove

Simply add a function `onRemove` to reveal a cross icon with a remove action (or other).

```jsx
function() {
  const [isHide, setHide] = React.useState(false)

  return isHide ? (
    <Text fontSize="sm" fontWeight="bold" color="danger.500" m={0}>
      Tag removed
    </Text>
  ) : (
    <Tag onRemove={() => setHide(true)}>Example of remove tag</Tag>
  )
}
```
