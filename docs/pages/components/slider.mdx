## Simple slider

```jsx
function () {
  const [value, setValue] = React.useState(null)

  const handleChange = (newValue) => {
    setValue(newValue)
  }

  return <Slider min={0} max={100} value={50} onChange={handleChange} />
}

```

## Differents uses

- `label` and `hint` props to add some information to your slider.
- `step` prop allow you to snap to predefined sets of values.
- `tooltip` prop to indicate the value being selected when dragging.

```jsx
<Slider min={0} max={100} value={50} label="Label" hint="hint" onChange={value => console.log(value)} />

<Slider min={0} max={100} value={0} step={10} onChange={value => console.log(value)} />

<Slider min={0} max={100} value={25} onChange={value => console.log(value)} tooltip />

<Slider min={0} max={100} value={50} onChange={value => console.log(value)} disabled />

```

## Types

There is 3 differents types for slider: `inline`, `left-field` and `right-field`.

```jsx
<Slider min={0} max={100} value={50} onChange={value => console.log(value)} type="inline" />

<Slider min={0} max={100} value={50} onChange={value => console.log(value)} type="left-field" />

<Slider min={0} max={100} value={50} onChange={value => console.log(value)} type="right-field" />
```

## Values

You can specify a range of values that will add a mark for each of them.

```jsx
<Slider min={0} max={100} value={50} onChange={value => console.log(value)} values={[0, 50, 100]} />

<Slider min={0} max={100} value={50} onChange={value => console.log(value)} values={[0, 25, 50, 75, 100]} />

<Slider min={0} max={100} value={50} step={10} onChange={value => console.log(value)} values={[0, 10, 20, 30, 40, 50, 60, 80, 70, 90, 100]} />
```

## borderSelectorColor

To match your background color, you should add a `borderSelectorColor`.

```jsx
<Stack backgroundColor="nude.100">
  <Slider
    min={0}
    max={100}
    value={50}
    onChange={value => console.log(value)}
    borderSelectorColor="nude.100"
  />

  <Slider.Range
    min={0}
    max={100}
    value={{ min: 25, max: 75 }}
    onChange={value => console.log(value)}
  />
</Stack>
```

## Range slider

The range slider contains almost all the props of the slider, so the usage of it is quite similar.

```jsx
 <Slider.Range min={0} max={100} value={{min: 25, max: 75}} onChange={value => console.log(value)} tooltip />

 <Slider.Range min={0} max={100} value={{min: 20, max: 80}} label="Label" hint="hint" onChange={value => console.log(value)} />

 <Slider.Range value={{min: 0, max: 50}} min={0} max={100} onChange={value => console.log(value)} values={[0, 50, 100]} />

 <Slider.Range value={{min: 25, max: 75}} min={0} max={100} step={25} onChange={value => console.log(value)} values={[0, 25, 50, 75, 100]} />

 <Slider.Range min={0} max={100} value={{min: 10, max: 90}} onChange={value => console.log(value)} disabled />
```

## Range Types

There is only 2 differents types for range slider: `inline` and `fields`.

```jsx
<Slider.Range value={{min: 10, max: 90}} min={0} max={100} onChange={value => console.log(value)} type="inline" />

<Slider.Range value={{min: 25, max: 75}} min={0} max={100} onChange={value => console.log(value)} type="fields" />
```
