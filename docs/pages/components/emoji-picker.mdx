# EmojiPicker

<tagversion version={version} name={name} component={component} />

EmojiPicker is a field element that extends [Popover](https://www.welcome-ui.com/components/popover) and [react-window](https://github.com/bvaughn/react-window) to efficiently render large list of images.  
It takes `value` & `onChange` props, its value being an emoji wrapped with colon, using the same pattern as Slack. The data is from [emoji-data](https://github.com/iamcal/emoji-data).

## Default

The most basic emoji picker needs `useEmojiPicker()`, `<EmojiPicker.Trigger />` and `<EmojiPicker />`.  
It will show a predefined list of emojis

```jsx
function Default() {
  const emojiPicker = useEmojiPicker()
  const [emoji, setEmoji] = useState()

  return (
    <>
      <EmojiPicker.Trigger as={Button} {...emojiPicker}>
        {emoji || 'Open Emoji Picker'}
      </EmojiPicker.Trigger>
      <EmojiPicker onChange={setEmoji} {...emojiPicker} />
    </>
  )
}
```

## Basic

```jsx
function Basic() {
  const emojiPicker = useEmojiPicker()
  const [emoji, setEmoji] = useState()

  return (
    <>
      <EmojiPicker.Trigger as={Button} {...emojiPicker}>
        {emoji || 'Open Emoji Picker'}
      </EmojiPicker.Trigger>
      <EmojiPicker onChange={setEmoji} {...emojiPicker}>
        <EmojiPicker.Tab name="Basic">
          <EmojiPicker.BasicList />
        </EmojiPicker.Tab>
      </EmojiPicker>
    </>
  )
}
```

## Custom

```jsx
function Custom() {
  const emojiPicker = useEmojiPicker()
  const [emoji, setEmoji] = useState()

  return (
    <>
      <EmojiPicker.Trigger as={Button} {...emojiPicker}>
        {emoji || 'Open Emoji Picker'}
      </EmojiPicker.Trigger>
      <EmojiPicker onChange={setEmoji} {...emojiPicker}>
        <EmojiPicker.Tab name="Basic">
          <EmojiPicker.BasicList />
        </EmojiPicker.Tab>
        <EmojiPicker.Tab name="Animals">
          <EmojiPicker.List
            emojis={constants.EMOJIS.filter(emoji => emoji.category === 'Animals & Nature')}
          />
        </EmojiPicker.Tab>
      </EmojiPicker>
    </>
  )
}
```
