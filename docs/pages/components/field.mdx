# Field

<tagversion version={version} name={name} component={component} />

We recommend using [React-hook-form](https://react-hook-form.com/) or [Formik](https://github.com/jaredpalmer/formik) with these components. If so, use `Field` for your fields (as in the examples below). The examples below give you an overview of common props (e.g. `size`, `disable`, `required` etc.)

You can find props for each field component in the other `Forms` pages.

```jsx
function () {
  const ref = React.useRef(null)

  const handleClick = () => {
    ref.current.focus()
  }

  return (
    <>
      <Field label="I'm a field with a ref on my child component">
        <InputText ref={ref} />
      </Field>
      <Button mt="md" onClick={handleClick}>Click me</Button>
    </>
  )
}
```

```jsx
function () {
  const [state, setState] = React.useState('')

  const handleChange = (event) => {
    setState(event.target.value + 'a')
  }

  return (
    <Field label="I'm a field with a state">
      <InputText onChange={handleChange} value={state} />
    </Field>
  )
}
```

```jsx
function () {
  const [state, setState] = React.useState(false)

  const handleChange = (event) => {
    setState(s => !s)
  }

  return (
    <Field label="I'm a field with a state" id="yolo">
      <Checkbox onChange={handleChange} checked={state} />
    </Field>
  )
}
```

## Refs

You can create a `ref`\* and pass it to a component to access the underlying DOM element.

Note: `ref`s can be created with either `React.createRef()` or `React.useRef(null)` for functional components or a callback ref if using a React `class`.

```jsx
function() {
  const InputRef = useRef(null)
  const nakedInputRef = useRef(null)
  const fileUploadRef = useRef(null)
  const buttonRef = useRef(null)
  const linkRef = useRef(null)

  const handleNormalClick = () => {
    InputRef.current.focus()
  }

  const handleNakedClick = () => {
    nakedInputRef.current.focus()
  }

  const handleFileUploadClick = () => {
    fileUploadRef.current.focus()
  }

  const handleButtonClick = () => {
    console.debug('<Button> ref', buttonRef)
  }

  const handleLinkClick = () => {
    console.debug('<Link> ref', linkRef)
  }

return (
    <Form>
      <Field label="Normal field" mb="sm">
        <InputText
          name="lastName"
          onChange={() => console.log('changed!')}
          placeholder="Placeholder"
          ref={InputRef}
        />
      </Field>
      <Label>Naked field</Label>
      <InputText name="naked" ref={nakedInputRef} mb="sm" />
      <Field label="Complex field" mb="sm">
        <FileUpload
          name="avatar"
          onChange={() => console.log('changed!')}
          ref={fileUploadRef}
        />
      </Field>
      <Box mt="xl" mb="sm">
        Click a button below to focus the relevant field above
      </Box>
      <Button onClick={handleNormalClick} ml={20}>
        Normal field
      </Button>
      <Button onClick={handleNakedClick} ml={20}>
        Naked field
      </Button>
      <Button onClick={handleFileUploadClick} ml={20}>
        File upload field
      </Button>
      <Box mt="xl" mb="sm">
        Click the button below to read the button's ref in the console
      </Box>
      <Button ref={buttonRef} onClick={handleButtonClick}>
        Test button
      </Button>
      <Box mt="xl" mb="sm">
        Click the link below to read the link's ref in the console
      </Box>
      <Link ref={linkRef} onClick={handleLinkClick}>
        Test link
      </Link>
    </Form>
  )
}
```
