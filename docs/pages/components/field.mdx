## About

We recommend using [React-hook-form](https://react-hook-form.com/) or [Formik](https://github.com/jaredpalmer/formik) with these components. If so, use `Field` for your fields (as in the examples below). The examples below give you an overview of common props (e.g. `size`, `disable`, `required` etc.)

You can find props for each field component in the other `Forms` pages.

## Label

Use `label` to add an Label to your field

```jsx
<Field label="Label">
  <InputText placeholder="Placeholder" />
</Field>
<Field label="Label for a checkbox">
  <Checkbox />
</Field>
```
## Hint

Use `hint` to add an hint to your field (it can be overload by a variant - see below)

```jsx
<Field hint="A hint">
  <InputText placeholder="Placeholder" />
</Field>
<Field hint="A hint" label="Label for a checkbox">
  <Checkbox />
</Field>
```

## Variants

Use `warning`, `error`, `info` or `success` properties to add a variant status on your fields. The label, hint or border color are set by magic ðŸª„

```jsx
<Field label="Label" error="An error">
  <InputText placeholder="Placeholder" />
</Field>
<Field error="An error" label="Label for a checkbox">
  <Checkbox />
</Field>
```

## Disabled

Add `disabled` on Field component

```jsx
<Field label="Label" disabled>
  <InputText placeholder="Placeholder" />
</Field>
<Field label="Label for a checkbox" disabled>
  <Checkbox />
</Field>
```

## Required

Add `required` on Field component

```jsx
<Field label="Label" required>
  <InputText placeholder="Placeholder" />
</Field>
<Field required label="Label for a checkbox">
  <Checkbox />
</Field>
```

## Refs

```jsx
function () {
  const inputRef = React.useRef(null)

  const handleClick = () => {
    inputRef.current.focus()
  }

  return (
    <>
      <Field label="I'm a field with a ref on my child component">
        <InputText ref={inputRef} />
      </Field>
      <Button mt="md" onClick={handleClick}>Focus InputText</Button>
    </>
  )
}
```
