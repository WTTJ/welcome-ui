## Variants

### Basics

```jsx row
<ButtonLink href="#basics">Primary</ButtonLink>
<ButtonLink href="#basics" variant="secondary">Secondary</ButtonLink>
<ButtonLink href="#basics" variant="tertiary">Tertiary</ButtonLink>
<Box backgroundColor="nude.700" display="inline" px="sm" py="xxl">
  <ButtonLink href="#basics" variant="tertiary-negative">Tertiary negative</ButtonLink>
</Box>
<ButtonLink href="#basics" variant="quaternary">Quaternary</ButtonLink>
```

## Formats

Use size property with `xs` `sm` `md`(default) `lg` or `xl`.

```jsx row
<ButtonLink href="#formats" size="xs">Tiny</ButtonLink>
<ButtonLink href="#formats" size="sm">Small</ButtonLink>
<ButtonLink href="#formats">Medium</ButtonLink>
<ButtonLink href="#formats" size="lg">Large</ButtonLink>
<ButtonLink href="#formats" size="xl">XLarge</ButtonLink>
```

## Disabled

All disabled buttons have the same style.

```jsx
<ButtonLink href="#disabled" variant="primary-danger" disabled>
  Disabled
</ButtonLink>
```

## Shape

You can set a `cirle` or `square` shape with `w` & `h` to have buttons pefect for icons.

```jsx row
<ButtonLink href="#shape" shape="square" size="xs">
  <EarthIcon size="xs" />
</ButtonLink>
<ButtonLink href="#shape" shape="circle" size="xs">
  <EarthIcon size="xs" />
</ButtonLink>
```

## Width

You can add `w` property from [xstyled](https://xstyled.dev/docs/width/).

```jsx
<ButtonLink href="#width" w="100%">full width</ButtonLink>
<ButtonLink href="#width" w={1 / 2}>half width</ButtonLink>
```

## Radius

You can add `borderRadius` property from [xstyled](https://xstyled.dev/docs/border-radius/).

```jsx row
<ButtonLink href="#radius">default</ButtonLink>
<ButtonLink href="#radius" borderRadius="10px">10px</ButtonLink>
```
