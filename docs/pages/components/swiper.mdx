## With custom pagination items

By providing the optional `renderPaginationItem` you can customise the defaults pagination bullets and render your own component. This callback takes a `props` param which is an object containning:

- `idx`: the current pagination item
- `pageIdx`: the currently selected slide (which you can use to show the active state for your pagination items)
- `onClick`: callback to pass to your custom bullet

```jsx
function() {
  const swiper = useSwiper({
    renderPaginationItem: ({idx,  active, onClick, pageIdx }) => (
      <Swiper.Bullet w={50} h={50} key={idx} active={idx === pageIdx} onClick={onClick} />
    )
  })

  return (
    <Swiper h={400} {...swiper}>
      <Swiper.Slide textAlign="center">
        <img
          style={{ maxWidth: '100%', maxHeight: '100%' }}
          src="https://images.unsplash.com/photo-1564460549828-f0219a31bf90?ixlib=rb-1.2.1&ixid=eyJhcHBfaWQiOjEyMDd9&auto=format&fit=crop&w=1350&q=80"
        />
      </Swiper.Slide>
      <Swiper.Slide textAlign="center">
        <img
          style={{ maxWidth: '100%', maxHeight: '100%' }}
          src="https://images.unsplash.com/photo-1575489272413-cb506258027e?ixlib=rb-1.2.1&ixid=eyJhcHBfaWQiOjEyMDd9&auto=format&fit=crop&w=634&q=80"
```
