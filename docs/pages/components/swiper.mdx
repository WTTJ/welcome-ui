## Slides per view

Providing `slidesPerView` with `[1, 1, 2]` will show 1 slide per view on mobile and tablet and 2 slides per view on desktop.

```jsx
function() {
  return (
    <Swiper h={400} slidesPerView={[1, 1, 2]} >
      <img
        style={{ maxWidth: '100%', maxHeight: '100%', objectFit: 'cover' }}
        src="https://images.unsplash.com/photo-1564460549828-f0219a31bf90?ixlib=rb-1.2.1&ixid=eyJhcHBfaWQiOjEyMDd9&auto=format&fit=crop&w=1350&q=80"
```

## Autoplay

Providing `autoplay` lets the slides advance automatically. Pass `duration` to control the amount of time each slide shows for.

```jsx
function() {
  return (
    <Swiper h={400} autoplay loop >
      <img
        style={{ maxWidth: '100%', maxHeight: '100%', objectFit: 'cover' }}
        src="https://images.unsplash.com/photo-1564460549828-f0219a31bf90?ixlib=rb-1.2.1&ixid=eyJhcHBfaWQiOjEyMDd9&auto=format&fit=crop&w=1350&q=80"
```

## With pagination

By providing the optional `withPagination` with `[true, true]`, you can deactivate the pagination on mobile/tablet and on desktop.

```jsx

  return (
    <Swiper h={400} withPagination={[true, true]}>
```
