## Usage

Is the InputText component with password conditions to show or hide the text.

```jsx
function () {
  const [value, setValue] = React.useState('')

  const handleChange = (event) => {
    setValue(event.target.value)
  }

  return <PasswordInput placeholder="enter your password" name="password" onChange={handleChange} value={value} />
}
```

## With Field

Example with a Field

```jsx
function () {
  const [value, setValue] = React.useState('')

  const handleChange = (event) => {
    setValue(event.target.value)
  }

  return (
    <Field label="Password" required>
      <PasswordInput placeholder="enter your password" name="password2" onChange={handleChange} value={value} />
    </Field>
  )
}
```
