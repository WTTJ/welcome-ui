## Size

Use size property with `xs`, `sm`, `md`, `lg` `xl` or `xxl`.

```jsx row
<XIcon size="xs" />
<XIcon size="sm" />
<XIcon />
<XIcon size="lg" />
<XIcon size="xl" />
<XIcon size="xxl" />
```

## Color

Icons inherit the color from their parent.

```jsx row
<XIcon color="blue" size="xl" />
<XIcon color="orange" size="xl" />
<XIcon color="green" size="xl" />
```
