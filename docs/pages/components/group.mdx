This works only with the `Button` component for now â€” more examples will be added soon.

### Button

```jsx
<Group>
  <Button>First</Button>
  <Button>Second</Button>
  <Button>Third</Button>
  <Button>Last</Button>
</Group>
```

## Overload

`Group` overloads the child components properties.

```jsx
<Group variant="tertiary" size="xs">
  <Button>First</Button>
  <Button variant="secondary">Second</Button>
  <Button variant="tertiary">Third</Button>
  <Button variant="tertiary">Last</Button>
</Group>
```
