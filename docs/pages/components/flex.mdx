## Usage

With the Flex component use shorthand props:

- `alignItems` is `align`
- `flexBasis` is `basis`
- `flexDirection` is `direction`
- `flexGrow` is `grow`
- `flexShrink` is `shrink`
- `flexWrap` is `wrap`
- `justifyContent` is `justify`

```jsx
<Flex justify={{ xs: 'center', md: 'space-between' }} align="center" wrap="wrap">
  <Box backgroundColor="secondary-blue" w={50} m="sm" h={50} />
  <Box backgroundColor="secondary-blue" w={50} m="sm" h={50} />
  <Box backgroundColor="secondary-red" w={50} m="sm" h={50} />
  <Box backgroundColor="secondary-red" w={50} m="sm" h={50} />
  <Box backgroundColor="secondary-green" w={50} m="sm" h={50} />
</Flex>
```
