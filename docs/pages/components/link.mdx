## About

A link component who can get `as` property to pass custom linker.

This component is a child from `UniversalLink` component.

## Variants

Can be `primary` by default or `secondary`

```jsx
<Box>
  <Link href="#random-link-primary">primary link</Link>
</Box>
<Box>
  <Link href="#random-link-secondary" variant="secondary">
    secondary link
  </Link>
</Box>
Example with multiple lines
<Box>
  <Link href="#random-link-multiple-lines" display="inline" variant="primary">
    Ipsum ipsam fugiat doloribus sit hic ducimus dolorem Aperiam unde adipisci deserunt corporis
    neque commodi. Quasi odit esse molestias reprehenderit incidunt Velit non expedita atque
    exercitationem enim! Quidem quasi nihil.
  </Link>
</Box>
```

### Disabled

Add `disabled` property

```jsx
<Link disabled href="#">
  disabled link
</Link>
```

## What is underlined by default?

`Link` will add an underline to text nodes, `Text` components, `span` tags, components with `as="span"` or `tagName="span"` & children with `data-wui-link` prop

```jsx
<Box>
  <Link href="#">
    <Avatar
      name="jungle"
      size="lg"
      src="https://avatars3.githubusercontent.com/u/13100706?s=200&v=4"
    />
    This is a text node
  </Link>
</Box>
<Box>
  <Link href="#">
    <Avatar name="jungle" size="lg" />
    <span>This is a text in a span tag</span>
  </Link>
</Box>
<Box>
  <Link href="#">
    <Avatar name="jungle" size="lg" />
    <Text>This is a text in a Text component</Text>
  </Link>
</Box>
<Box>
  <Link href="#">
    <Avatar name="jungle" size="lg" />
    <div>This is a text in a div tag (won't add an underline)</div>
  </Link>
</Box>
<Box>
  <Link href="#">
    <Avatar name="jungle" size="lg" />
    <Box as="span">This is a Box with as="span" prop</div>
  </Link>
</Box>
<Box>
  <Link href="#">
    <Avatar name="jungle" size="lg" />
    <div data-wui-link>This is a div with data-wui-link prop</div>
  </Link>
</Box>
```

## Security

For a target `_blank` we add `rel='noopener noreferrer'`

```jsx
<Link href="#" target="_blank">
  with a target blank
</Link>
```

## External link

`isExternal` add an `<ExternalLink />` icon, and add special effect on hover.

```jsx
<Link href="#" target="_blank" isExternal>
  External link
</Link>
```

## With a Text component

By default link is set to medium on fontWeight, you can force inherit to get the parent style.

```jsx
<Text variant="h5">
  A text with h5{' '}
  <Link href="#" fontWeight="inherit" target="_blank">
    variant
  </Link>
</Text>
```

## Usage with React Router

<code>
{`import { Link as RouterLink } from 'react-router-dom'

<Link as={RouterLink} to="/jobs">
  Jobs
</Link>`}
</code>
