# MarkdownEditor

<tagversion version={version} name={name} component={component} />

We use [react-simplemde-editor](https://github.com/RIP21/react-simplemde-editor) as a base with the following changes:

- We've added simple styles around it to keep it consistent with our other form fields
- We've made the toolbar icons more flexible (see below)
- The toolbar is sticky for when your Markdown gets really long

## Toolbar

You can send an array of toolbar items with the shape:

`{ name: 'bold', icon: <i className="fa fa-bold" />, action={item => { // Do something with item }}, title: 'Bold ðŸŒ´' }`.

For example, in the toolbar below we're using icons from:

- [FontAwesome](https://fontawesome.com/icons?d=gallery) (bold, italic)
- [Material Design Iconic Font](https://zavoloklom.github.io/material-design-iconic-font/icons.html): (strikethrough, link)
- our own [Icons](/icons): (code, quote)
- and plain text (emoji)

Note: to use FontAwesome or Material Design icons (or any other 3rd-party icon font), you'll have to include the font stylesheet somewhere in your app.

```jsx
<Form initialValues={{}}>
  <ConnectedField
    component={MarkdownEditor}
    label="Markdown Editor"
    name="welcome"
    placeholder="Placeholder"
    toolbar={[
      { name: 'bold', icon: <i className="fa fa-bold" />, title: 'Bold ðŸŒ´' },
      { name: 'italic', icon: <i className="fa fa-italic" /> },
      { name: 'divider' },
      { name: 'strikethrough', icon: <i className="zmdi zmdi-format-strikethrough-s" /> },
      { name: 'link', icon: <i className="zmdi zmdi-link" /> },
      { name: 'divider' },
      { name: 'code' },
      { name: 'quote', title: 'Quote ðŸŒ´' },
      { name: 'divider' },
      { name: 'emoji', icon: 'ðŸ‘Œ' }
    ]}
  />
</Form>
```

## Actions

You can add some actions on the bottom of the textarea with `actions` childrens. We calculate automatically the height of actions.

```jsx
<Form initialValues={{}}>
  <ConnectedField
    component={MarkdownEditor}
    label="Markdown Editor"
    name="welcome-actions"
    placeholder="Placeholder"
    actions={
      <>
        <Button variant="quaternary" shape="square" size="sm">
          <HashtagIcon />
        </Button>
        <Button variant="quaternary" shape="square" size="sm">
          <MentionIcon />
        </Button>
        <Button variant="quaternary" shape="square" size="sm">
          <AttachmentIcon />
        </Button>
      </>
    }
  />
</Form>
```
