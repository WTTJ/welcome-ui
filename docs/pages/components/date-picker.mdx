## About

It is based on the [react-datepicker](https://github.com/Hacker0x01/react-datepicker) library.

_Note_: Pass a value of `null` if you don't want the default value of `Date.now()`.

## Usage

Pass a date for the value

```jsx
function () {
  const [value, setValue] = React.useState(Date.now())

  const handleChange = newValue => {
    setValue(newValue)
  }

  return (
    <Field label="Date">
      <DatePicker name="welcome" onChange={handleChange} value={value} />
    </Field>
  )
}
```

## Props from react-datepicker

Simply add on DatePicker the props from [react-datepicker](https://github.com/Hacker0x01/react-datepicker)

```jsx
function () {
  const [value, setValue] = React.useState(Date.now())

  const handleChange = newValue => {
    setValue(newValue)
  }

  return (
    <Field label="Date">
      <DatePicker
        dateFormat="MMM dd yyyy"
        name="welcome2"
        maxDate={new Date()}
        popperProps={{ zIndex: 50 }}
        yearDropdownItemNumber={30}
        onChange={handleChange}
        value={value}
      />
    </Field>
  )
}
```

## Sizes

Use size propety with option:
- `xs` (24px)
- `sm` (32px)
- `md` (40px - default)
- `lg` (48px)

```jsx
function () {
  const [value, setValue] = React.useState(Date.now())

  const handleChange = newValue => {
    setValue(newValue)
  }

  return (
    <Stack spacing="xl">
      <DatePicker name="welcome3" size="xs" value={value} onChange={handleChange} />
      <DatePicker name="welcome3" size="sm" value={value} onChange={handleChange} />
      <DatePicker name="welcome3" value={value} onChange={handleChange} />
      <DatePicker name="welcome3" size="lg" value={value} onChange={handleChange} />
    </Stack>
  )
}
```

## Locales

You can change your locale by pass locale property from date-fns.

In this exemple we import **import dateFR from 'date-fns/locale/fr'**

```jsx
function () {
  const [value, setValue] = React.useState(Date.now())

  const handleChange = newValue => {
    setValue(newValue)
  }

  return (
    <Field label="Date">
      <DatePicker locale={dateFR} name="welcome4" value={value} onChange={handleChange} />
    </Field>
  )
}
```
