## About

Notification component with [toasted-note](https://toasted-notes.netlify.com) and 2 components:

- Growl
- Snackbar

## useToast

Use `useToast` hook to show the toast element. Example below with a custom component.

```jsx
function() {
  const toast = useToast()

  const Notification = () => (
    <Box
      backgroundColor="light-900"
      borderWidth="1px"
      borderStyle="solid"
      borderColor="nude-300"
      borderRadius="sm"
      padding="sm"
      color="dark-900"
    >
      Lorem ipsum dolor sit amet
    </Box>
  )

  return (
    <Button onClick={() => toast(<Notification />)}>
      Show Toast
    </Button>
  )
}
```

## Options

The `toast` function takes a component to be displayed and an `options` object:

- **position** : one of `top-left`, `top`, `top-right`, `bottom-left`, `bottom` (default) or `bottom-right`
- **duration** : before closing element, in ms, with default `7000`. Can also be set to `null` if not automatic close your element.

```jsx
function() {
  const toast = useToast()

  const Notification = () => (
    <Box
      backgroundColor="light-900"
      borderWidth="1px"
      borderStyle="solid"
      borderColor="nude-300"
      borderRadius="sm"
      padding="sm"
      color="dark-900"
    >
      Lorem ipsum dolor sit amet
    </Box>
  )

  return (
    <Stack direction="row">
      {['top-left', 'top', 'top-right', 'bottom-left', 'bottom', 'bottom-right'].map(position => (
        <Button
          key={position}
          onClick={() =>
            toast(
              <Notification />,
              { position, duration: 6000 }
            )
          }
        >
          {position}
        </Button>
      ))}
    </Stack>
  )
}
```

## Supplied components

You can create your own notification component to display in the Toast or one of our default components: `Snackbar` and `Growl`.

In general :

- `Snackbar` is used to display a temporary, information-only notification, and
- `Growl` is used to display a more important notification with an action (that requires the user to close the notification)

## Custom close function

You can add your custom function when user click on close button

```jsx
function() {
  const toast = useToast()

  function onClose() {
    alert('Toast hidden!')
  }

  return (
    <Button
      onClick={() =>
        toast(
          <Toast.Growl>
            <Toast.Title>
              Lorem ipsum dolor sit amet
            </Toast.Title>
            Class aptent taciti sociosqu ad
          </Toast.Growl>,
          { onClose, position: 'top-right' }
        )
      }
    >
      Open
    </Button>
  )
}
```

## Without close button on Growl or Snackbar

You can choose not to show a close button on Growl or Snackbar with the `hasCloseButton` prop (defaults to `true`).

```jsx
function() {
  const toast = useToast()

  function onClose() {
    alert('Toast hidden!')
  }

  return (
    <Stack direction="row">
      <Button
        onClick={() =>
          toast(
            <Toast.Snackbar  hasCloseButton={false}>
                Lorem ipsum dolor sit amet
            </Toast.Snackbar>
          )
        }
      >
        Snackbar with no close button
      </Button>
      <Button
        onClick={() =>
          toast(
            <Toast.Growl hasCloseButton={false}>
              <Toast.Title>
                Lorem ipsum dolor sit amet
              </Toast.Title>
              Class aptent taciti sociosqu ad
            </Toast.Growl>,
            { position: 'top-right' }
          )
        }
      >
        Growl with no close button
      </Button>
    </Stack>
  )
}
```
