## Usage

```jsx row
<Badge>{1}</Badge>
<Badge>New</Badge>
<Badge shape="square" variant="primary">1</Badge>
<Badge shape="square" variant="primary">New</Badge>
```

## Sizes

Use size property with `sm` or `md` (default)

```jsx row
<Badge size="sm">sm</Badge>
<Badge>md</Badge>
```

## Variants

Use variant property with `default` (default) or `primary`

```jsx row
<Badge>default</Badge>
<Badge variant="primary">primary</Badge>
```

## Disabled

Use disabled property

```jsx row
<Badge disabled>default</Badge>
<Badge disabled variant="primary">primary</Badge>
```

## Shapes

Use shape property with `circle` (default) or `square`

```jsx row
<Badge>{1}</Badge>
<Badge>{100}</Badge>
<Badge shape="square">{1}</Badge>
<Badge shape="square">{100}</Badge>
```

## withNumberAbbreviation

If a number is higher than 99, we replace this number by 99+, simply add property `withNumberAbbreviation`. *The children need to be a number!*

Variant can be `default` (default) or `primary`

```jsx row
<Badge withNumberAbbreviation>{100}</Badge>
```
