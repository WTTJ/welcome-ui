## With icons

```jsx
<Form initialValues={{}}>
  <ConnectedField
    component={Picker}
    options={[
      {
        element: ({ selected }) => (
          <Shape shape="circle" width={30} backgroundColor={selected ? 'nude.200' : 'unset'}>
            <EditIcon color="dark.900" />
          </Shape>
        ),
        value: 'edit'
      },
      {
        element: ({ selected }) => (
          <Shape shape="circle" width={30} backgroundColor={selected ? 'nude.200' : 'unset'}>
            <TwitterIcon color="dark.900" />
          </Shape>
        ),
        value: 'twitter'
      },
      {
        element: ({ selected }) => (
          <Shape shape="circle" width={30} backgroundColor={selected ? 'nude.200' : 'unset'}>
            <PencilIcon color="dark.900" />
          </Shape>
        ),
        value: 'pencil'
      }
    ]}
    name="icon"
    label="Icons"
    required
  />
</Form>
```
